# 🚀 Fullstack Web3 DApp

## Tech Stack

- **Backend**: FastAPI + Beanie + PyMongo Async API + Strawberry GraphQL
- **Frontend**: React + Vite + TypeScript + Ant Design Pro + Apollo Client + Web3 (wagmi/viem)
- **Database**: MongoDB
- **Storage**: MinIO (S3-compatible)
- **Infra**: Docker + Nginx
- **CI/CD**: GitHub Actions
